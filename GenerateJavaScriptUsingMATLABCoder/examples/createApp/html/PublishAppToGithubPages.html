Publish App to GitHub Pages

This example shows how to publish an app created with the Generate JavaScript using MATLAB Coder add-on toolbox to GitHub® Pages. For more information on GitHub Pages, see GitHub Pages.

Set Up the App Project

1. In the MATLAB® current folder browser, navigate to a folder where you can create and store your app project and files. Create the app project by entering this code in MATLAB command line. Specify a name and the directory for your app.
proj = webcoder.setup.project("My New App","Directory",pwd);
2. Optionally, you can add callbacks to your app project and edit the index.html file. Build the app project by entering this code in the command line.
proj = openProject(pwd);
webcoder.build.project(proj);
3. The resulting app is the set of 4 files located in the dist folder.

Set Up Source Control and Publish

Follow these steps to push a new build or any updated build of your app to GitHub® pages.
1. Configure and share your project to your GitHub repository. For more information on sharing MATLAB projects, see Share Project on GitHub.
2. At the MATLAB command prompt, enter this code to push the dist folder to GitHub.
!git add dist && git commit -m "Initial dist subtree commit"
3. Push the dist subtree to GitHub Pages.
!git subtree push --prefix dist origin gh-pages
