Setup for Generate JavaScript Using MATLAB Coder

Introduction

The Generate JavaScript Using MATLAB Coder add-on requires you to install the Emscripten software development kit (SDK) on your computer. The Emscripten SDK uses a group of tools to convert C/C++ code into optimized JavaScript. For more information on Emscripten, see About Emscripten.

Install Emscripten and Setup WebCoder Tools

1. Download and install the Emscripten SDK onto your computer. For details, see Download and install. Carefully follow the instructions provided on the Emscripten website, as different operating systems require specific extra steps to install all necessary third-party tools.
Note: This add-on does not modify your system environment variables. On Windows, you can install the Emscripten SDK without using the global (-g) flag.
2. In MATLAB®, for OS X® and Linux® operating systems, verify MATLAB uses a BASH shell environment.
if ismac || isunix
webcoder.setup.verifyShellEnvironment;
end
3. Provide the path to the Emscripten SDK root folder. The folder is typically named emsdk. Enter this command to navigate to the folder by opening a file explorer.
emsdkRootDir = webcoder.setup.getEmscriptenRootDirectory;
4. Verify the installation of all tools required by this add-on and Emsripten.
webcoder.setup.verifyThirdPartyTools(emsdkRootDir);
5. Save the location of the third-party tools into this add-on.
webcoder.setup.saveThirdPartyToolPaths(emsdkRootDir);
6. Create and register the Emscripten toolchain with MATLAB Coder™.
webcoder.setup.createAndRegisterToolchain;
